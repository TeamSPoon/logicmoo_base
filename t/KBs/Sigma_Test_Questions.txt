:: ==================================================================
:: Sigma Test Questions: a file of HPKB TQs in English:  and KIF formulations.
:: These questions depend on information in the HPKBCrisis context (including also HPKBScenarioConflict 
:: and others).
:: The questions were answered by Cyc, and the questions have been answered by XSB queries run over 
:: corresponding XSB data.
:: For the Cyc answers to these queries, see the appropriate TQIdent justification files.
:: Test results should be recorded.
:: The (KIF) information needed to answer these questions resides in the file, HPKBCrisis_Context_TestInfo.kif
:: Additional HPKBCrisis data, in KIF format, resides in HPKBCrisis_Context.kif. These files should
:: be amalgamated, when testing.
:: The XSB .P information needed to answer these question resides in the file, HPKBCrisis_Context_TestInfo.P
:: This file (and other KIF files) can be 'cyclified' using the PERL tool, cycl.pl, for input to Sigma 
:: (remove ":: KIF:", before use -- cycl.pl will remove the comments, here prefixed with "::").
:: N.B. Some of the TQE and F queries below have a natural HiLog formulation in XSB.
