InfraEngine Example RDF Documents

  • Examples Provided with the System
    • rdf/vehicles.rdf
      The vehicles example after the W3C's RDF Primer. Its is discussed in Guided Tour: RDF Schema Browsing with InfraEngine.
    • rdf/blocks.rdf
      A blocks world planning example. Its is discussed in Guided Tour: Planning with InfraEngine.
    • rdf/table.rdf
      A planning example that demonstraints use of query parameters and arithmetic constraints. It is discussed in Guided Tour: RDF Schema Browsing with InfraEngine.

  • Meta-Level Schemas of the System
    • infraengine_schema.rdf
    • config_schema.rdf
      The schema of the config.rdf InfraEngine configuration file.

  • The Forth RDF Schema Registry
    A large collection of RDF ontologies that have been cleaned-up to be parseable.
    • Original Page: http://139.91.183.30:9090/RDF/Examples.html
    • Local Copy
      A script to create a local copy can be found at bin/download-forth-rdf-registry.sh.
