pdt_find_clauses(+Context, +Signature, -DefFile, -Clauses)
Find clauses of a goal.

(This predicate is used by the 'Open Declaration'-feature of the PDT)

Parameters:
ContextModule- The module in the context of which the goal is to be resolved
Signature- an unqualified predicate signature of the form Name>/Arity>
DefFile- the file in which the clauses where found
Clauses- a list of annotated clause terms.
